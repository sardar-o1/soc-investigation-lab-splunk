# SOC Lab Dashboard

## Overview

A Splunk dashboard was created to provide a centralized view of the simulated attack lifecycle, from initial access through payload execution. The dashboard helps analysts quickly visualize attacker activity, correlate events across multiple log sources, and understand the progression of the attack.
